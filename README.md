# ormolu.el

A formatter for Haskell source code.

# Usage

With [use-package](https://github.com/jwiegley/use-package/) + [quelpa](https://framagit.org/steckerhalter/quelpa):

```elisp
(use-package ormolu
 :quelpa
 (ormolu
  :fetcher github
  :repo "vyorkin/ormolu.el"))
```
